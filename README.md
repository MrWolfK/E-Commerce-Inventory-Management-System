# E-Commerce-Inventory-Management-System

An Inventory Management System was created with the Java 
language. This is a desktop application and makes use of the standard
Java Swing libraries to achieve a user-friendly, visual data
management experience without the help of external databases.
The main design of the system is based on Object-Oriented
Programming techniques, which separate the item data, core logic,
and graphical interface. The program implements an Array List to
keep track of a growing list of products, and data management is done
by using basic loops and control structures. Among the features of
the system are custom error handling, a search function to find items
by name or ID, and sorting mechanics that let the user organize items
by price or alphabet. Besides that, the project has implemented an
easy-to-use interface with text fields, a data table, and working
buttons so that the user can add, update, or delete products on the fly.
This work is a perfect example of basic programming concepts such
as class inheritance, custom exceptions, arrays, and graphical user
input handling.
